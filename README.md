# Connect-4-AI-Agent

**An AI Agent for Connect 4 Game using Adversial Searching Algorithms:**

- **Minmax** Algorithm.
- **Alpha-Beta Pruning Minmax** Algorithm.
- **Expectiminmax** Algorithm.
